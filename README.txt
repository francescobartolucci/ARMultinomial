R code to estimate the models of the paper titled: 

 "A multivariate statistical approach to predict 
 COVID-19 count data with epidemiological interpretation and uncertainty quantification"

# Authors: Bartolucci, F., Pennoni, F. and Mira, A. #

This is the list of files to estimate the models: 

- BPM_models_GITHUB.R load data and estimate the proposed models 

- multinomAR_mcmcFB.R proposed MCMC algorithm to estimate the models with the Multinomial distribution

- dirmultAR_mcmcFB.R proposed MCMC algorithm to estimate the models with the Dirichlet-Multinomial distribution

- Italy.csv: contains the observed data for Italy

The file to obtain tables and figures of the paper:

- BPM_results.R
